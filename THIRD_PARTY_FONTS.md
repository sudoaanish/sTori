# Third-party fonts

sTori bundles the Latin variable-font builds listed below through Fontsource.
Each family is distributed under the SIL Open Font License 1.1. The complete
license text is retained in each installed package's `LICENSE` file.

| Family | Project | Package |
| --- | --- | --- |
| Merriweather | https://github.com/SorkinType/Merriweather | `@fontsource-variable/merriweather` |
| Literata | https://github.com/googlefonts/literata | `@fontsource-variable/literata` |
| Source Serif 4 | https://github.com/adobe-fonts/source-serif | `@fontsource-variable/source-serif-4` |
| Newsreader | https://github.com/productiontype/Newsreader | `@fontsource-variable/newsreader` |

When release packaging is added, include this notice and the corresponding OFL
license text in the installed application's third-party notices.
